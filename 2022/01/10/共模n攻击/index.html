共模n攻击

发表于2022-01-10|更新于2022-01-21|Crypto
|阅读量:

例题

攻击条件

c1,e1,c2,e2,n => m

已知n，同一份密文m用e1和e2两个公钥分别加密得到c1和c2两份密文

原理

已知 c1,e1,c2,e2,n

由此可得：

又因为gcd(e1,e2)=1
所以 $e_{1}s_{1}+e_{2}s_{2}=1$ 肯定能找到一组解

结论为：

证明如下：
$(c_{1}^{s_{1}} *c_{2}^{s_{2}}) \% n $

$=((m^{e_{1} } \% n)^{s_{1}}*(m^{e_{2} } \% n)^{s_{2}}) \% n$

$=(((m^{e_{1} } \% n)^{s_{1}} \% n)*((m^{e_{2} } \% n)^{s_{2}}) \%n) \% n$

$=((m^{e_{1}s_{1} } \% n)*(m^{e_{2}s_{2} } \%n)) \% n$

$=m^{e_{1}s_{1} + e_{2}s_{2} } \% n$

$=m \% n$

$=m$

变形

前面所说的攻击手法都是建立在e1，e2互素的基础上的。那么如果e1，e2不互素呢？

不妨设$\gcd(e_{1} , e_{2} ) =a$，那么由扩展欧几里得算法，求得的一组s1,s2就不是使等式右边为1了，而是$s_{1}e_{1}+s_{2}e_{2}=a $

上面的推导过程到$=m^{e_{1}s_{1} + e_{2}s_{2} } \% n$之后就是等于$m^{a}%n$，所以最后要开a次方根。
