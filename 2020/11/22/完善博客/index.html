发表于2020-11-22|更新于2020-11-26
|阅读量:

我的主题：butterfly
附上链接代码：git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
pug和 stylus 的渲染器（必备）：npm install hexo-renderer-pug hexo-renderer-stylus –save

语言：
修改站点配置文件 _config.yml
default(en)
zh-CN (简体中文)
zh-TW (繁体中文)
头像：
修改主题配置文件实现，在配置文件里找到以下语句，修改即可
avatar:
img: /img/avatar.png
网站背景：
在url后面的括号里填上路径

该主题有个独特的网站副标题，可以自动切换语句:
把enable和effect后面都设为true，注意冒号后面的空格，在source后面填写来源

关于上传博客
常用命令：
（1） hexo new [layout] <br> [layout]可以为以下三种：<br> 参数名 功能 文章路径<br> post 新建博文 source/_posts<br> page 新建页面 source<br> draft 草稿 source/_drafts<br> （2） hexo publish [layout] <title><br> 用于发布草稿，也可以直接将博文从_drafts文件夹拖入_posts文件夹

上传图片：
1）把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true

2）在hexo目录下执行这样一句话npm install hexo-asset-image –save用于安装插件
3）当创建文章时，会自动在文章同一目录下生成同名文件夹，将图片放入
假设我要插入的图片名字为1.jpg，放在博客主目录source文件夹下的XXX文件夹里。
使用hexo相对路径插入图片
可能会有一定的问题，这里附上一个链接，里面有解决方案
https://blog.csdn.net/qq_44875386/article/details/105510258
